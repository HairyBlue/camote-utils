@rhynel/camote

A comprehensive TypeScript utility library for string and number formatting

TypeScript Zero Dependencies Modern ES2018+

Installation

npm install @rhynel/camote

Features

Number Formatting

  • Human-readable numbers (K, M, B)
  • Comma-separated numbers
  • Percentage formatting
  • Ordinal numbers
  • File size formatting
  • Currency formatting

String Manipulation

  • Case conversions
  • URL-friendly slugs
  • Word counting
  • String padding
  • Template formatting
  • String reversal

Development Features

  • Full TypeScript support
  • 58+ unit tests
  • Zero dependencies
  • Modern ES2018+ compatible
  • Comprehensive documentation

Quick Example

import { humanReadableNumber, formatCurrency } from '@rhynel/camote';

// Format numbers
console.log(humanReadableNumber(1500000));  // "1.5M"
console.log(formatCurrency(1234.56));       // "$1,234.56"

// Format strings
import { slugify, capitalize } from '@rhynel/camote';
console.log(slugify("Hello World!"));      // "hello-world"
console.log(capitalize("hello"));          // "Hello"

Documentation

Number Formatting

humanReadableNumber

humanReadableNumber(value: number): string

Converts a number to a human-readable format with K, M, B suffixes.

Parameter Type Description
value number The number to format 
humanReadableNumber(1234);     // "1.2K"
humanReadableNumber(1500000);  // "1.5M"
humanReadableNumber(1000000000); // "1.0B"

formatCurrency

formatCurrency(value: number, currency?: string, locale?: string): string

Formats a number as currency with proper separators and symbol.

Parameter Type Description
value number The amount to format
currency string Currency code (e.g., 'USD', 'EUR')
locale string Locale identifier (e.g., 'en-US') 
formatCurrency(1234.56);                    // "$1,234.56"
formatCurrency(1234.56, 'EUR', 'de-DE');   // "1.234,56 â‚¬"

String Manipulation

slugify

slugify(text: string): string

Converts a string into a URL-friendly slug.

Parameter Type Description
text string The text to convert to a slug 
slugify("Hello World!");      // "hello-world"
slugify("What's Up?");        // "whats-up"

toCamelCase

toCamelCase(text: string): string

Converts a string to camelCase format.

Parameter Type Description
text string The text to convert 
toCamelCase("hello-world");   // "helloWorld"
toCamelCase("Hello World");   // "helloWorld"
