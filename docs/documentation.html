camote-utils Documentation

v1.0.0

Installation

Install camote-utils using npm:

npm install camote-utils

Basic Usage

Import the functions you need:

import { formatNumber, formatCurrency } from 'camote-utils';

Number Formatting

Converts a number into a human-readable string with appropriate suffixes (K, M, B, T).

Syntax

function formatNumber(
    value: number,
    options?: {
        precision?: number;
        format?: 'short' | 'long';
    }
): string

Parameters

  • value: The number to format
  • options (optional):
    • precision: Number of decimal places (default: 2)
    • format: Output format - 'short' (K, M, B) or 'long' (thousand, million, billion)

Examples

formatNumber(1234);       // "1.23K"
formatNumber(1234567);    // "1.23M"
formatNumber(1234567890); // "1.23B"

// With options
formatNumber(1234, { precision: 1 });           // "1.2K"
formatNumber(1234, { format: 'long' });         // "1.23 thousand"
formatNumber(1234567, { format: 'long' });      // "1.23 million"

Currency Formatting

Formats a number as a currency string using the specified locale and currency code.

Syntax

function formatCurrency(
    value: number,
    currency: string = 'USD',
    locale: string = 'en-US'
): string

Parameters

  • value: The number to format
  • currency: ISO 4217 currency code (default: 'USD')
  • locale: BCP 47 language tag (default: 'en-US')

Examples

formatCurrency(1234.56);                    // "$1,234.56"
formatCurrency(1234.56, 'EUR');             // "€1,234.56"
formatCurrency(1234.56, 'JPY', 'ja-JP');    // "¥1,235"
formatCurrency(1234.56, 'GBP', 'en-GB');    // "£1,234.56"

String Manipulation

Converts a string to a slug, camel case, kebab case, or snake case.

Syntax

function slugify(str: string): string
function camelCase(str: string): string
function kebabCase(str: string): string
function snakeCase(str: string): string

Examples

slugify('Hello World');       // "hello-world"
camelCase('hello world');    // "helloWorld"
kebabCase('hello world');    // "hello-world"
snakeCase('hello world');    // "hello_world"

Pluralization

The pluralize function helps you convert singular words to their plural form based on a count. It handles common English pluralization rules and supports custom plural forms. 

// Basic pluralization
pluralize("cat", 1);              // "cat"
pluralize("cat", 2);              // "cats"
pluralize("baby", 2);             // "babies"
pluralize("box", 2);              // "boxes"

// Custom plural forms
pluralize("child", 2, "children"); // "children"
pluralize("person", 2, "people");  // "people"

Parameters

  • word (string) - The singular form of the word
  • count (number, optional) - The count to determine pluralization
  • customPlural (string, optional) - Custom plural form

Returns

Returns the pluralized or singular word based on count.

TypeScript Usage

This library is written in TypeScript and includes type definitions. No additional @types packages are needed.

import { formatNumber, CurrencyOptions } from 'camote-utils';

// Type checking works out of the box
const options: CurrencyOptions = {
    currency: 'USD',
    locale: 'en-US'
};

const formatted = formatCurrency(1234.56, options);

Browser Support

This library is compatible with all modern browsers and Node.js environments. It's built using ES2018+ features and includes necessary polyfills for older environments.

<script src="https://unpkg.com/camote-utils"></script>
<script>
    const { formatNumber } = window.camoteUtils;
    console.log(formatNumber(1234567)); // "1.23M"
</script>
